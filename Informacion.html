1 FORMULARIO ESTUDIANTE

Este formulario tiene la funcionalidad de recuperar información de los estudiantes almacenados, permitiendo manipular los datos de cada registro. Para ello brinda las opciones de carga de estudiantes, cuya función es mostrar cada uno de los registros de los estudiantes; actualizar estudiante, este permite actualizar los datos de un registro; ingresar estudiante, este sirve para agregar un nuevo estudiante en la base de datos; y, por último, eliminar asignaturas, dicha opción es para suprimir un registro almacenado en la base de datos.


2 FORMULARIO ASIGNATURA

Dentro de nuestro formulario podrá encontrar las especificaciones de cada una de las Asignaturas registradas dentro de un esquema de bases de datos, nuestro formulario esta diseñado para hacer varias funciones con respecto a la información presentada, detalles de funcionalidad:

  1. Carga de asignaturas: A primera vista, encontrara todos los registros que forman parte de nuestro esquema de datos.
  2. Actualizar asignaturas: Los campos por cada asignatura pueden ser actualizados.
  3. Ingresar asignaturas: Tenemos la opción de agregar nuevas asignaturas a nuestro esquema de datos.
  4. Eliminar asignaturas: Los registros que ya no queremos que formen parte de nuestro esquema podrán ser eliminados.
